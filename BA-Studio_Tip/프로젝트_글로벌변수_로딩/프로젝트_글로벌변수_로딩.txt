
1. 액티비티 EXCEL/CreateInstance

2. 액티비티 EXCEL/GetWorksheetAsDictionary
   Name   dictionary_0
   Target excel_0
   path {{ R.ConfigPath + R.ConfigFile }}    #  예) R.ConfigPath: C:\{프로젝트명}\Config\    R.ConfigFile: Config.xlsx
   sheet    0   # 시트 index 왼쪽부터 0 
   keyIdx   1   # 키 로 입력될 항목 엑셀 A열 0, B열 1, C열 2, ...
   valueIdx 2   # 값 으로 입력될 항목 엑셀 A열 0, B열 1, C열 2, ...

3. 액티비티 BuiltIn/Process
   G["d"] = {}  # 변수 초기화
   
4. 액티비티 BuiltIn/Process
   G["d"].update(dictionary_0)    # 글로벌 변수 생성
   
5. 액티비티 EXCEL/Exit

위 5개 액티비티들을 차례대로 연결 하면 된다. 


다른 Task에서 글로벌 변수 사용법
액티비티:     COMMON/LogWrite
Filename:  {{ G["d"]["LogPath"] }}  
Format:    %Y-%m-%d %H:%M:%S
Ext:       log
Message:   {{ G["LOG"] }}

# Filename의 값을 절대경로\파일명을 입력해야 함. 예) C:\Project\Log\ProjectLog 
위 처럼 입력하면 실제로는
C:\Project\Log\ProjectLog_20201228 13.log
위 처럼 로그파일이 생성되고 파일 내용으로 Message 가 입력 된다.
