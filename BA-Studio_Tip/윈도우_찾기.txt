윈도우 찾기


Findwindow() 로 메인 창을 찾는다.

win32gui 모듈에 Findwindow() 가 구현되어있다.



예)

import win32gui
import win32con
hwnd = win32gui.FindWindow(None, "메모장")

창 최대화
win32gui.ShowWindow(hwnd, win32con.SW_MAXIMIZE)

print(hwnd)
