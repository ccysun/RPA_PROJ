
▶ format 함수 사용법
format 함수는 중괄호 {, } 안에 포매팅을 지정하고 format 함수의 인자로 값들을 넣습니다.

>>> a = 2
>>> b = 3
>>>
>>> s = '구구단 {} x {} = {}'.format(a, b, a * b)
>>> print(s)
구구단 2 x 3 = 6
